    静止している観測者，振動数$f_0$\tanni{Hz}の音を発する音源S，音を反射する壁が図のように一直線上に並んでおり，壁はこの直線に対して垂直に置かれている。Sと壁はその直線上を移動できるものとする。音速を$V$\tanni{m/s}とする。
        \begin{enumerate}
            \item 壁は固定して，音源Sを左へ速さ$v$\tanni{m/s}で動かす場合，
                \begin{enumerate}
                    \item Sから観測者に直接到達する音の振動数はいくらか。
                    \item 壁で反射して観測者に到達する音の振動数はいくらか。
                    \item 観測者には1秒間に$n_1$回のうなりが聞こえた。Sの速さ$v$はいくらか。
                \end{enumerate}
            \item 音源Sを固定して，壁を左へ速さ$u$\tanni{m/s}で動かす場合，
                \begin{enumerate}
                    \item 壁で反射して観測者に到達する音の振動数はいくらか。
                    \item 観測者には1秒間に$n_2$回のうなりが聞こえた。壁の速さ$u$はいくらか。
                \end{enumerate}
        \end{enumerate}
